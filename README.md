Aplicação para fazer o controle financeiro.

## Instalação

Configure o .env com as seguintes variáveis de ambiente:

> SECRET_KEY  
> HOSTS  
> CORS_ALLOWED_ORIGINS  
> CSRF_TRUSTED_ORIGINS  
> 
> MYSQL_DATABASE  
> MYSQL_HOST  
> MYSQL_PORT  
> DEBUG  
> MYSQL_USER  
> MYSQL_PASSWORD  

Instale a aplicação:

> make install

## Rodando a aplicação

> make run
